NutriGaze is an AI-powered image classification platform designed to detect whether fruits and vegetables are healthy or rotten using deep learning technology.

Our system uses a trained Convolutional Neural Network (VGG16-based model) to analyze uploaded images and accurately classify produce into 28 categories, including both healthy and rotten classes.

  • ✔ Real-time fresh vs rotten detection using deep learning.
  • ✔ 28-class classification covering multiple fruits and vegetables.
  • ✔ Helps reduce food waste through early spoilage identification.
  • ✔ Simple web interface for easy image upload and instant results.

Built using Python, Flask, and a pre-trained VGG16 transfer learning model, NutriGaze combines artificial intelligence with practical food quality inspection. This project demonstrates how machine learning can be applied to solve real-world agricultural and food safety challenges.
